  • Responsive Resume Builder

    A web-based application that allows users to easily create, preview, and download professional resumes in real time.     Key Features:

    • User-friendly form for entering all resume sections
    • Multiple template options for different styles
    Technologies Used: HTML, CSS, JavaScript

  • Basic Calculator

    A user-friendly desktop calculator built with Python and Tkinter. This project supports basic arithmetic operations, trigonometric functions (sin, cos, tan), square root, percentage, and mathematical constants (π, e). The calculator features a clean interface, error handling, and allows users to perform both simple and scientific calculations efficiently.

    Key Features:

    • Addition, subtraction, multiplication, division, modulus
    • Trigonometric functions: sin, cos, tan
    • Square root, parentheses, decimal point
    • Percentage, π (Pi), e (Euler's Number)
    • Scientific and engineering notation
    Technologies Used: Python, Tkinter
